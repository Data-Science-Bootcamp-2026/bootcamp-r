**WORKSHOP PART 1:**

1. Go to the [book](https://data-science-bootcamp-2026.github.io/bootcamp-r/)
2. At the top left, click the GitHub icon (the cat)
3. Near the top right of the GitHub page, click the green **<> Code** button
4. Click **Download ZIP**
5. Unzip it

Let's create an R project:

6. Open RStudio
7. Click the button on the top right to open a dropdown menu
8. Click **New Project...**
9. Because we are creating a project within the repository we just downloaded, click **Existing Directory**
10. To set the project root, find where you saved the downloaded, unzipped repo
    - Beware that when you unzipped it, it might create a parent folder of the same name -- you want to set the project root to be the `bootcamp-r-main` folder that directly houses all the folders

Now work through the prepared exercise:

11. Open `analysis/02_create_tableone.R` and work through it line by line, following the documentation
